# Project Overview
This repository contains the build scripts and instructions for the Nethunter project targeting specific devices.

# Device Info
The project supports various Android devices. Make sure to check the device compatibility before proceeding with the build.

# Prerequisites
- A compatible Android device
- Linux-based OS
- Required packages (such as git, build-essential, etc.)
- Sufficient disk space and memory

# Build Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/kasukuM/note3-nethunter-build.git
   cd note3-nethunter-build
   ```
2. Install the required dependencies:
   ```bash
   sudo apt-get install <dependencies>
   ```
3. Run the build script:
   ```bash
   ./build.sh
   ```
4. Follow further instructions displayed during the build process.
